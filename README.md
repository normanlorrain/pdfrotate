A Windows program to rotate PDF pages.  A mashup of [Flet](https://flet.dev/) and [PyMuPDF ](https://pymupdf.readthedocs.io/en/latest/document.html)


To generate the installer, [WiX toolset](https://wixtoolset.org/).
